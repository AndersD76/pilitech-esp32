Perguntas Frequentes - PILI TECH
1. O que é o PILI TECH?
Sistema de telemetria IoT que monitora 8 sensores do tombador em tempo real, conta ciclos, horímetro e envia dados para a nuvem.
2. Como acessar localmente?
Conecte ao WiFi "PILI-TECH" (senha: numero de serie do equipamento) e acesse 192.168.4.1 no navegador.
3. Como é contado um ciclo?
O ciclo é contado quando o sensor de 0° passa de ATIVO para INATIVO (plataforma começa a subir). O tempo padrão é 20 minutos.
4. Como funciona o tracking do ciclo?
Portão fecha = início do ciclo. Cada sensor tem seu tempo ativo medido (ON→OFF). Portão abre = fim do ciclo. O dashboard mostra a duração de cada sensor e o tempo total.
5. Para que serve o horímetro?
Conta as horas de operação para programar manutenções preventivas a cada 2000 horas de uso.
6. Os dados ficam salvos offline?
Sim! Sem internet, os dados são salvos a cada 15 min e sincronizados automaticamente ao reconectar.
7. O que significa o alerta de Moega/Fosso?
Indica que o fosso está cheio e precisa ser esvaziado. O sistema emite alerta visual e sonoro.
8. Como iniciar o sistema?
Na aba Sistema, com WiFi conectado, o técnico deve digitar a senha e clicar em START para iniciar operação.
